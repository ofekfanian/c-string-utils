# String Utils

A menu-driven C program with three utility functions.

## Features

- **Matrix border sum** — reads a matrix and sums all border elements clockwise
- **String concatenation** — manual implementation of `strcat`
- **Substring search** — manual implementation of `strstr`, returns index and character

## How to compile and run
```bash
gcc main.c functions.c -o string_utils
./string_utils
```

## What I practiced
- Pointers and pointer arithmetic in C
- 2D arrays and matrix traversal
- Modular code with header files
